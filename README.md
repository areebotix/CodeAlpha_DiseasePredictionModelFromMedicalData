# ❤️ Heart Disease Prediction using Machine Learning

## 📌 Overview
This project predicts the likelihood of heart disease using machine learning models trained on the UCI Heart Disease dataset.

## 📊 Dataset
- Source: UCI Machine Learning Repository
- Features: Age, cholesterol, blood pressure, etc.
- Target: Presence of heart disease

## ⚙️ Workflow
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training (LR, SVM, RF, XGBoost)
- Hyperparameter Tuning
- Model Evaluation

## 🏆 Best Model
XGBoost achieved the best performance with ~83% test accuracy after reducing overfitting.

## 📈 Evaluation Metrics
- Accuracy
- Confusion Matrix
- ROC Curve
- Feature Importance

## 🚀 How to Run
1. Install requirements:
pip install -r requirements.txt
2. Run notebook or prediction script:
python src/predict.py

## 📌 Key Insight
Medical prediction requires balancing accuracy and recall to reduce false negatives.
