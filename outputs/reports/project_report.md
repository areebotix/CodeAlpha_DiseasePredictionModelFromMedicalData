# Heart Disease Prediction Project Report

## 1. Objective
To build a machine learning system that predicts the presence of heart disease using clinical and demographic data.

## 2. Dataset
Dataset sourced from the UCI Machine Learning Repository. It contains patient attributes such as age, cholesterol, blood pressure, and heart-related indicators.

## 3. Methodology
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training (Logistic Regression, SVM, Random Forest, XGBoost)
- Hyperparameter Tuning
- Model Evaluation

## 4. Best Model
XGBoost performed best with approximately 83% test accuracy after reducing overfitting.

## 5. Evaluation Metrics
- Accuracy
- Confusion Matrix
- ROC Curve
- Feature Importance

## 6. Conclusion
The model demonstrates good predictive ability but shows slight overfitting, which was reduced using regularization and subsampling techniques.
